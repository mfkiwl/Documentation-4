class Kratos::PointerVectorSet< TDataType, TGetKeyType, TCompareType, TEqualType, TPointerType, TContainerType >

PointerVectorSet is a sorted associative container like stl set but using a vector to store pointers to its data.

PointerVectorSet is a sorted associative container like stl set but using a vector to store pointers its data. Many methods are copied from boost ptr_container library. There is modification to make it capable to work with shared pointers.

This Container unlike the boost one does not free the memory by itself and relies on using of counted pointers or manual deleting.
