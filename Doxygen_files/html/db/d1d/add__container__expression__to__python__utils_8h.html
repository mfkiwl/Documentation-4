Forbids the casting by defining the forbidden type methods.

pybind11 casts to the method arguments types which ever is passed from python side. If the types are not matching, then a new object is made from copying and then casting. For large vectors this is an expensive operation. Therefore, this macro is used to forbid the casting and throw an error if an unsupported numpy array is passed to the function.
