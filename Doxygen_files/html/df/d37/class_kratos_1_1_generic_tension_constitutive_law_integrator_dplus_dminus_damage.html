Detailed Description

template<class TYieldSurfaceType>
class Kratos::GenericTensionConstitutiveLawIntegratorDplusDminusDamage< TYieldSurfaceType >

: This object integrates the predictive stress using the isotropic the d+d- damage theory

The definitions of these classes is completely static, the derivation is done in a static way The damage integrator requires the definition of the following properties:

  • SOFTENING_TYPE: The fosftening behaviour considered (linear, exponential,etc...)

    TYieldSurfaceTypeThe yield surface considered
