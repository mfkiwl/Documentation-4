Detailed Description

This method fills rDestinationModelPart using data obtained from rOriginModelPart. The elements of rDestinationModelPart part use the same connectivity (and id) as in rOriginModelPart but their type is determined by rReferenceElement. Both ModelParts will share the same nodes.
