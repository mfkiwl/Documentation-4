Detailed Description

template<unsigned int TDim, unsigned int TNumNodes = TDim>
class Kratos::WallConditionDiscontinuous< TDim, TNumNodes >

Implements a wall condition for the monolithic formulation.

It is intended to be used in combination with ASGS and VMS elements or their derived classes and the ResidualBasedPredictorCorrectorVelocityBossakSchemeTurbulent time scheme, which supports slip conditions. This condition will add a wall stress term to all nodes identified with SLIP==true (in the nodal flags). This stress term is determined according to the wall distance provided as Y_WALL.

See also
ASGS2D,ASGS3D,VMS,ResidualBasedPredictorCorrectorVelocityBossakSchemeTurbulent
