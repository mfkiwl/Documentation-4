Detailed Description

template<class TPointType>
class Kratos::Hexahedra3D8< TPointType >

An eight node hexahedra geometry with linear shape functions.

The node ordering corresponds with: v 3-------—2 |\ ^ |\ | \ | | \ | \ | | \ | 7---—+—6 | | +– |– | -> u 0—+—--1 | \ | \ \ | \ | \ \ | | w | 4-------—5
