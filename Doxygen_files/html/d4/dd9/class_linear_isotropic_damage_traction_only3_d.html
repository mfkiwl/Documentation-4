Detailed Description

Defines a damage with hardening/softening constitutive law in 3D.

This material law is defined by the parameters:

  • YOUNG_MODULUS
  • POISSON_RATIO
  • YIELD_STRESS
  • INFINITY_YIELD_STRESS
  • ISOTROPIC_HARDENING_MODULUS
